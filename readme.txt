Enable DCDN URLs for your static assets such as images, CSS or JavaScript files.

== Description ==

Simply integrate a distributed Content Delivery Network (dCDN) into your WordPress site.

= What it does? =
The dCDN Engine plugin has been developed to link your content to the CDN URLs.

= Features =
* Link assets to load from a CDN
* Set included directories
* Define exclusions (directories or extensions)
* Enable or disable for HTTPS


= System Requirements =
* PHP >=5.6
* WordPress >=4.6
